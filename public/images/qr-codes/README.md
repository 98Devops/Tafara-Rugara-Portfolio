# QR Codes

This directory contains QR codes for quick contact access.

## Files

- `whatsapp-qr.svg` - QR code linking to WhatsApp contact (https://wa.me/263777553271)
- `linkedin-qr.svg` - QR code linking to LinkedIn profile (https://www.linkedin.com/in/tafara-rugara-0627b819b/)

## Generation

These QR codes are generated using the `scripts/generate-qr-codes.js` script.

To regenerate:
```bash
node scripts/generate-qr-codes.js
```

## Usage

These QR codes are displayed in the AutomationSystems component for the flagship AI-Powered Delivery Health & Operations Intelligence System project, allowing visitors to quickly connect via WhatsApp or LinkedIn.
