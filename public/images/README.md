# Open Graph Images

This directory should contain the following Open Graph images for optimal SEO:

## Required Images (1200x630 pixels)

- `og-home.jpg` - Home page Open Graph image featuring Tafara Rugara
- `og-capabilities.jpg` - What I Do page image showcasing technical capabilities
- `og-projects.jpg` - Projects page image highlighting featured projects
- `og-experience.jpg` - Experience page image showing professional background
- `og-contact.jpg` - Contact page image with contact information theme
- `profile.jpg` - Professional profile photo for structured data

## Image Guidelines

- **Dimensions**: 1200x630 pixels (Facebook/LinkedIn recommended)
- **Format**: JPG or PNG
- **File size**: Under 1MB for optimal loading
- **Content**: Should include Tafara's name, title, and relevant visual elements
- **Text**: Large, readable fonts (minimum 24px)
- **Branding**: Consistent with portfolio design (dark theme, blue/purple gradients)

## Alternative Fallback

If custom images are not available, the system will fall back to a default Open Graph image or generate one programmatically.
